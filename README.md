# LDAP Authentication with Node.js

A simple node.js lib to authenticate against an LDAP server.

## Pre-Requisite

- Must chage the **config** part at **trf3ad.js**

```sh 
$ yarn add activedirectory
```


## Usage

```sh 
$ node trf3ad.js
```
